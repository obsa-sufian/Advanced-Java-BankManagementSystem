# BankManagementSystem2

Java Swing bank management system (Advanced Java course project).

## Build & Run

- Requires JDK 8+.
- Build using Ant: `ant` (project contains `build.xml`).
- Run: `java -cp build/classes MainApplication` or run via your IDE.

## Notes

- Configure `resources/config.properties` for your database connection.
- Database schema is available at `resources/database.sql`.
